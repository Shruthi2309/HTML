HEADING TAG

There are 6 predefined heading tags in HTML which is used to display text in different font size and in bold face.

1. H1 tag defines the text with latgest bold font.

2. H2 tag defines the text with font lesser than h1 tag.

3. H3 tag defines the text with font lesser than h2 tag.

4. H4 tag defines the text with font lesser than h3 tag.

5. H5 tag defines the text with font lesser than h4 tag.

6. H6 tag defines the text with font lesser that h5 tag.
